# Worktile Swift SDK

基于官方 Rest API 使用 Swift 2.0 为 Worktile 封装的一套 SDK，方便 Cocoa 开发者使用 Swift 进行开发。

> 计划支持 iOS 和 OS X，目前先针对 iOS 平台进行开发完善，后续会支持 OS X 平台

### 开发环境：

- Xcode 7.0 beta
- Swift 2.0
- iOS8 +

### 实现功能

- [x] OAuth2 登录授权
- [ ] 获取用户信息
- [ ] 团队
- [ ] 项目
- [ ] 任务组
- [ ] 任务
- [ ] 日程
- [ ] 文件
- [ ] 话题
- [ ] 文档

> 这里可以随时查看开发进度，也欢迎有兴趣的开发者来添砖加瓦！
> 
> 有任何问题也请您让我知道，分分钟给修复它！
